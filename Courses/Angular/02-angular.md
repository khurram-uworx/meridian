This section introduces Angular as a framework and prepares trainees to work effectively within its ecosystem.

Learning objectives:

1. Environment setup and Angular CLI
- Install Node.js and Angular CLI and create a new project.
- Understand Angular project structure.
- Generate components, services, and modules with CLI commands.

2. Components, templates, and directives
- Create and integrate new components.
- Use interpolation, property binding, event binding, and two-way binding.
- Apply structural directives (*ngIf, *ngFor) and attribute directives (ngClass, ngStyle).

3. Dependency injection
- Explain dependency injection and its role in Angular.
- Create and inject services into components.
- Explain provider scopes (root, module, component).

4. Routing and navigation
- Configure routes and navigate between pages.
- Use route parameters and query parameters.
- Build a simple multi-page app with navigation.

5. Forms and user input
- Build template-driven and reactive forms.
- Add validation (required, min/max length, custom).
- Handle form state and validation errors.

6. HTTP and data handling
- Use HttpClient for GET/POST requests.
- Handle responses and errors.
- Display data in components using async patterns.

7. Testing and debugging
- Run Angular default tests (Jasmine/Karma).
- Write at least one basic component or service test.
- Use Angular DevTools for debugging.

8. Running Angular in Docker
- Run Angular locally in Docker.
- Explain image build and container-based run workflow.
- Explain benefits of containerized frontend development.

Completion criteria:

- Trainee builds and runs a simple Angular app locally and in Docker.
- Trainee implements navigation, a validated form, and HTTP data consumption.
- Trainee executes at least one unit test.
- Instructor validates functionality through review, Q&A, and demo.
